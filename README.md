# Zara Injury Law - Personal Injury Website

Professional personal injury law website for attorney Michael A. Zara.

## Deployment
This site is configured for deployment on Vercel. Connect this repository to Vercel for automatic deployment.

## Structure
- 40+ SEO-optimized static HTML pages
- Practice area pages with FAQ schema
- State-specific landing pages (CO, AZ, FL, GA, NC, WI)
- City-level pages for major markets
- Blog with legal content
- Full JSON-LD schema (Attorney, LocalBusiness, FAQPage, BreadcrumbList)
- sitemap.xml and robots.txt
